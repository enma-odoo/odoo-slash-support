# odoo-slash-support

**Author:** [Enock Maseru](https://github.com/enma-odoo)
**Version:** 1.0
**Status:** ⚠️ Unofficial Community Tool  

A lightweight Chrome extension for Odoo Business Analysts. Instantly route to database support pages and view the active server version right from your toolbar.

## ✨ Features
* **One-Click Routing:** Click the extension icon to instantly append `/_odoo/support` to your current Odoo database URL.
* **Live Version Badging:** Automatically detects and displays the server version (e.g., `18.0`, `19.2`) directly on the extension icon.

## 🚀 Installation & Usage
1. Clone the repository:
   ```bash
   git clone [https://github.com/enma-odoo/odoo-slash-support.git](https://github.com/enma-odoo/odoo-slash-support.git)
   ```
2. Navigate to `chrome://extensions/` in your browser and enable **Developer mode** (top right).
3. Click **Load unpacked** and select the cloned folder.
4. **Usage:** Pin the extension to your toolbar. Click it when on an Odoo database to route to support, or simply glance at it to check the current version.
